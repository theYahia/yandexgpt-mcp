# /yandexgpt-summarize -- Суммаризация текста через YandexGPT

## Используемые инструменты

| Инструмент | Что делает |
|------------|------------|
| `chat` | Чат с YandexGPT для суммаризации |

## Алгоритм

1. Получи текст от пользователя
2. Если модель не указана -- по умолчанию yandexgpt-lite
3. Вызови `chat` с системным промптом: "Ты -- ассистент-суммаризатор. Выдели ключевые тезисы и сделай краткое резюме. Формат: 3-5 пунктов + итог в 1 предложении."
4. Покажи суммаризацию

## Формат ответа

```
## Суммаризация YandexGPT

**Модель**: yandexgpt-lite

### Ключевые тезисы:
- <тезис 1>
- <тезис 2>
- ...

### Итог:
<краткое резюме>
```

## Примеры

```
/yandexgpt-summarize <вставь длинный текст>
/yandexgpt-summarize <текст> yandexgpt
```
