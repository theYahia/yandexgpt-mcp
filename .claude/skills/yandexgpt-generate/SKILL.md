# /yandexgpt-generate -- Генерация текста через YandexGPT

## Используемые инструменты

| Инструмент | Что делает |
|------------|------------|
| `completion` | Синхронная генерация текста через YandexGPT |

## Алгоритм

1. Разбери запрос пользователя: текст, модель, температура
2. Если модель не указана -- по умолчанию yandexgpt-lite
3. Вызови `completion` с параметрами
4. Покажи результат с информацией о модели и использованных токенах

## Формат ответа

```
## Результат YandexGPT

**Модель**: yandexgpt-lite
**Ответ**: <текст ответа>
**Токены**: ввод: 15, вывод: 128, всего: 143
```

## Примеры

```
/yandexgpt-generate Напиши стихотворение о весне
/yandexgpt-generate Объясни квантовую механику yandexgpt 0.5
```
